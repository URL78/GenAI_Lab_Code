# 🚀 GenAI Lab Repository

This repository contains a collection of hands-on lab assignments focused on **Generative AI concepts**, including LLM usage, prompt engineering, testing, and practical implementations.

The goal of this repository is to explore **real-world applications of Generative AI** through structured experiments and coding exercises.

---

## 📌 Overview

This lab covers:

- Large Language Models (LLMs)  
- Prompt Engineering Techniques  
- Code Refactoring using LLMs  
- Test Case Generation  
- MCQ & Interview Question Generation  
- Retrieval-Augmented Generation (RAG)  
- Software Testing with `pytest`  
- Coverage Analysis
- Requirement Elicitation Technique
- Creat Google Chrome Extention
